---
title: DROP GROUP

---

Removes a database role.

<a id="section2"></a>

## Synopsis

<div id="sql_command_synopsis"></div>

```
DROP GROUP [IF EXISTS] <name> [, ...]
```

<a id="section3"></a>

## Description

`DROP GROUP` is an alias for [DROP ROLE](DROP_ROLE.md).

<a id="section5"></a>

## Compatibility

There is no `DROP GROUP` statement in the SQL standard.
